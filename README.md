# Agrobeso Website

Premium, mobile-first homepage for Agrobeso (authentic Ghanaian and West African food in South London), built with React + TypeScript + Vite + Tailwind CSS.

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Project structure

```txt
src/
  assets/
  components/
  data/
  lib/
  pages/
  styles/
```

## Setup

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Content editing

Restaurant content is centralized in:

- `src/data/restaurant.ts`

Before launch, update placeholder items such as:

- Unverified opening hours
- Unconfirmed branch phone numbers
- Delivery links
- Gallery placeholders with owner-approved photos
